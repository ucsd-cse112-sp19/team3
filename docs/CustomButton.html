    CustomButton

    Create a new CustomButton component with designated styles.

    Usage

    See CustomButton Demo for all listing examples.

    <custom-button property="value">
    Button Text
</custom-button>

    Properties:
    Name Type Description
    style string

    Same as style attribute in native CSS.
    text-color string

    Change color of the text within the button.
    background-color string

    Change background color of the button.
    border-color string

    Change the color of the border.
    font string

    Change the font of the text within the button.
    width string

    Change the width of the button.
    height string

    Change the height of the button.
    hover-text-color string

    Change the text color when hovering over the button.
    hover-background-color string

    Change the background color when hovering over the button.
    hover-border-color string

    Change the border color when hovering over the button.
    active-text-color string

    Change the text color when the button is being clicked.
    active-background-color string

    Change the background color when the button is being clicked.
    active-border-color string

    Change the border color when the button is being clicked.
    rounded boolean

    Make the button appear with rounded edges.
    circle boolean

    Make the button appear as a circle.
    disabled boolean

    Make the button unable to be clicked.
    href string

    Pass a link to the button.

    updateStyle()

    UPDATES THE STYLE BY TAKING USER ATTRIBUTES, INCLUDING 'STYLE' AND OUR OWN API OF SPECIAL ATTRIBUTES, AND PASSING THEM TO THE CSS FOR THE BUTTON. IF THE USER STYLES AN ATTRIBUTE (E.G. FONT COLOR) USING BOTH DIRECT CSS AND OUR CUSTOM ATTRIBUTE (TEXT-COLOR), IT PRIORITIZES CUSTOM ATTRIBUTES.
