    CoreHello

    A starter of customized web components.

    Usage

    See CoreHello Demo for all listing examples.

    <core-hello property="value">
    Display Text
</core-hello>

    Properties:
    Name Type Description
    style string

    Same as style attribute in native CSS.
    class string

    Same as class attribute in native CSS.
    lang string

    Set the language of display, default is English, other options: jp, es, fr
    rainbow boolean

    Enable rainbow color effect.
