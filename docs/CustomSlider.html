    CustomSlider

    new CustomSlider()

    Create a new CustomSlider component with designated styles.

    See CustomSlider Demo for all listing examples.

    Usage

    <custom-slider property="value"></custom-slider>

    Properties:
    Name Type Description
    min string

    Set the minimum value of the slider.
    max string

    Set the maximum value of the slider.
    step string

    Set the step size when moving the slider.
    value string

    Set the initial value of the slider.
    disabled boolean

    Make the slider unabled to be used.
    readonly boolean

    Set the input field as read-only.
    required boolean

    Set whether required for like a form (if thereâ€™s a form bounding this component).
    showinput boolean

    Enables a text box containing the current value of the slider.
    onchange string

    Callback function when value is updated.
    slider-class string

    Class attribute for the slider in native CSS.
    input-class String

    Class attribute for the input field in native CSS.
