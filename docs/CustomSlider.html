    CustomSlider

    new CustomSlider()

    Create a new CustomSlider component with designated styles.

    See CustomSlider Demo for all listing examples.

    Usage

    <custom-slider property="value"></custom-slider>

    Properties:
    Name Type Description
    disabled boolean

    Disables the slider, graying out the bar and preventing value change.
    size string

    S, M, or L options allow size customization.
    max number

    Sets the maximum value of the slider.
    min number

    Sets the minimum value of the slider.
    showInput boolean

    Enables a text box containing the current value of the slider.
    value number

    Sets the initial value of the slider.
