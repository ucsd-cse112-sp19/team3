    CustomPopover

    new CustomPopover()

    Create a new CustomPopover component with designated styles.

    See CustomPopover Demo for all listing examples.

    Usage

    <custom-popover property="value"> Popover Body Text </custom-popover>

    Properties:
    Name Type Description
    anchor string

    Name of the button to which the popover is anchored.
    color string

    Set the background color.
    font string

    Set the font type.
    header-text string

    Set the text content.
    placement string

    Place popover on the button's top, bottom, left, or right.
    style string

    Set a CSS style.
    textcolor string

    Set the text color.
