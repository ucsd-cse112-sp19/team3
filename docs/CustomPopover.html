    CustomPopover

    Create a new CustomPopover component with designated styles.

    Usage

    See CustomPopover Demo for all listing examples.

    <custom-popover property="value">
    Popover Text
</custom-popover>

    Properties:
    Name Type Description
    anchor string

    Id of button to which the popover is anchored.
    color string

    Set the background color.
    font string

    Set the font type.
    header-text string

    Set the text content.
    placement string

    Direction in which to place popover.
    style string

    Set a CSS style.
    textcolor string

    Set the text color.

    Examples
    <custom-button id='default-btn'>
     <custom-popover anchor='default-btn'>
         Popover Text
     </custom-popover>
     Click Me
</custom-button>
    Popover Text Click Me 
    <custom-button rounded id="color-btn">
     <custom-popover anchor="color-btn" color="hotpink" placement='right'>
         Popover Text
     </custom-popover>
     Click Me
</custom-button>
    Popover Text Click Me 
    <custom-button rounded id="font-btn">
     <custom-popover anchor="font-btn" color="hotpink"
                     placement='right' font="Courier New">
         Popover Text
     </custom-popover>
     Click Me
</custom-button>
    Popover Text Click Me 
    <custom-button rounded id="textcolor-btn">
     <custom-popover anchor="textcolor-btn" color="hotpink"
                     placement='left' text-color="black">
         Popover Text
     </custom-popover>
     Click Me
</custom-button>
    Popover Text Click Me 
    <custom-button class="btn-warning" id='btnWarning'>
     <custom-popover class='btn-warning' anchor='btnWarning'
                     placement='bottom'>
         Popover Text
     </custom-popover>
     Warning
</custom-button>
    Popover Text Warning
