    CustomPopover

    Create a new CustomPopover component with designated styles.

    Usage

    See CustomPopover Demo for all listing examples.

    <custom-popover property="value">
    Popover Text
</custom-popover>

    Properties:
    Name Type Description
    anchor string

    Id of button to which the popover is anchored.
    color string

    Set the background color.
    font string

    Set the font type.
    header-text string

    Set the text content.
    placement string

    Direction in which to place popover.
    style string

    Set a CSS style.
    textcolor string

    Set the text color.

    <custom-button id='default-btn'>
     <custom-popover anchor='default-btn'>
         Popover Text
     </custom-popover>
     Click Me
</custom-button>

     <custom-popover anchor="color-btn" color="hotpink" placement='right'>

    UPDATES THE STYLE BY TAKING USER ATTRIBUTES, INCLUDING 'STYLE' AND OUR OWN API OF SPECIAL ATTRIBUTES, AND PASSING THEM TO THE CSS FOR THE POPOVER. IF THE USER STYLES AN ATTRIBUTE (E.G. FONT COLOR) USING BOTH DIRECT CSS AND OUR CUSTOM ATTRIBUTE (TEXT-COLOR), IT WILL PRIORITIZE THE CUSTOM.
